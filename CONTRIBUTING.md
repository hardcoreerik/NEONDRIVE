# Contributing

NEONDRIVE is a solo development project. I'm not accepting pull requests or
code contributions at this time.

**Pull requests will be closed without merging.**

## Bug Reports

Bug reports are welcome. If you find a hardware issue, firmware crash, or
incorrect behavior, open an Issue with:

- Which board you're using (CYD 2.4 / 2.8 / 3.5, Tab5, Cardputer, etc.)
- What you expected to happen
- What actually happened
- Serial output if relevant

## Forks

You're welcome to fork this project and build on it for your own use.

## Why solo?

This is a personal lab tool I build and test on my own hardware. Accepting
outside code would require review, testing across boards I may not have, and
ongoing maintenance I can't commit to. Keeping it solo keeps it moving fast.

For security-related reports, see [SECURITY.md](SECURITY.md).
