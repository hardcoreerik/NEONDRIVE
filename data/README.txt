CYD PorkChop Port — Milestone D
-----------------------------------
What you get:
- Home menu UI (WiFi / Targets / Config / About)
- WiFi Scan screen with Scan/Stop + AP list + tap-to-select
- LittleFS-backed /config.json with in-UI editing + save
- Locked CYD baseline display init order:
  1) SPI.begin(14,12,13,15)
  2) tft.init()
  3) tft.setRotation(1)     (landscape, USB on RIGHT)
  4) tft.invertDisplay(true)
  5) use runtime dims tft.width()/tft.height()
  6) fill + 1px border

Touch mapping:
- Baked PRESET #5 (invertX only) for landscape USB-right.

How to build/upload (PowerShell)
--------------------------------
1) Open project folder
2) Build:
   pio run -e cyd
3) Upload firmware:
   pio run -e cyd -t upload
4) Upload LittleFS (uploads ./data to /):
   pio run -e cyd -t uploadfs
5) Serial monitor:
   pio device monitor -b 115200

Notes
-----
- TFT_eSPI is configured via include/User_Setup.h and forced by platformio.ini build_flags.
- /config.json will be created automatically on first boot if missing, but using uploadfs
  lets you start with known values immediately.
